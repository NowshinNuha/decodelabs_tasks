# Project 1: Social Media Content Planning - DecodeLabs

## 🎯 Project Overview
This repository contains the strategic 30-day content roadmap designed for DecodeLabs during my Digital Marketing Internship. The strategy utilizes a "Split Brain Ecosystem" approach to segment B2B trust acquisition from B2C audience engagement.

## 👥 Target Audience Segmentation
* **LinkedIn (Trust Engine):** The Buying Committee (CIOs, TPOs, and University Faculty).
* **Instagram (Engagement Layer):** End-Users (Students, Aspiring Coders, and Tech Enthusiasts).

## 📅 30-Day Psychological Funnel Summary
* **Week 1:** Problem Awareness (Addressing the Theory vs. Practice Gap)
* **Week 2:** Technical Value (Showcasing code execution and Vibe Coding snippets)
* **Week 3:** Proving Authority (Leveraging AICTE/MSME verification documents and raw UGC)
* **Week 4:** Conversion (High-value, low-friction action prompts)

## 📁 Files Included in this Repository
1. **content_matrix.md**: Contains the exact captions and visual concepts for the posts.
2. **operational_logbook.xlsx**: The scheduling matrix tracking platform, triggers, and metrics.

## 📊 Performance Tracking Strategy
Instead of monitoring vanity metrics like generic post likes, success for this strategy is measured via:
1. **LinkedIn:** Target profile visits from decision makers (HR, Training & Placement Officers).
2. **Instagram:** Conversion velocity and the quality of incoming structural inbound queries in direct messages.
